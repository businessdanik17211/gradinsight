## Структура новых таблиц БД

### profession_forecasts
- profession_name - название профессии
- category - категория (worker/employee/specialist)
- demand_level - уровень спроса (high/medium/low)
- forecast_year - год прогноза
- source - источник данных
- city - город
- description - описание
- related_specialties - связанные специальности

### profession_salaries
- profession_name - название профессии
- avg_salary - средняя зарплата
- min_salary - минимальная зарплата
- max_salary - максимальная зарплата
- vacancies_count - количество вакансий
- city - город
- year/month - период
- source - источник

### profession_analytics (VIEW)
Объединенное представление с рейтингом профессий

## Команды для работы

```bash
# Установка зависимостей
cd scripts/data-collection
npm install

# Запуск сбора прогнозов
npm run collect:forecasts

# Запуск сбора зарплат
npm run collect:salaries

# Запуск всего
npm run collect:all
```

## Импорт в БД

```bash
# Supabase CLI
supabase db push

# Или через psql
psql $SUPABASE_DB_URL -f scripts/data-collection/output/profession-forecasts-2026.sql
psql $SUPABASE_DB_URL -f scripts/data-collection/output/profession-salaries-2026.sql
```

- Данные о зарплатах основаны на маркетинговых исследованиях
