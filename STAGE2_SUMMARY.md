## Созданные файлы

### 1. Типы данных
- `src/types/professions.ts` - TypeScript типы для профессий
  - ProfessionAnalytics, ProfessionForecast, ProfessionSalary
  - Типы для фильтров и статистики
  - Переводы категорий и уровней спроса

### 2. Сервисный слой
- `src/services/professionService.ts` - API для работы с данными
  - getProfessions() - получение всех профессий с фильтрами
  - getTopProfessions() - топ профессий по рейтингу
  - getProfessionsByDemand() - фильтрация по спросу
  - getStats() - статистика по профессиям
  - И другие методы

### 3. React Hooks
- `src/hooks/useProfessions.ts` - Хуки для работы с данными
  - useProfessions() - основной хук для списка профессий
  - useTopProfessions() - хук для топа профессий
  - useProfessionByName() - хук для деталей профессии

### 4. UI Компоненты
- `src/components/cards/DemandIndicator.tsx` - Индикатор уровня спроса
- `src/components/cards/ProfessionCard.tsx` - Карточка профессии
- `src/components/sections/ProfessionsSection.tsx` - Секция на странице

### 5. Интеграция
- Обновлен `src/pages/Applicants.tsx` - добавлена секция профессий

## Функциональность

### Для абитуриентов доступно:
1. **Просмотр востребованных профессий 2026**
   - 83 профессии из официальных источников
   - Фильтрация по уровню спроса (высокий/средний/низкий)
   - Фильтрация по категориям (рабочие/служащие/специалисты)
   - Поиск по названию

2. **Информация о каждой профессии**:
   - Уровень востребованности (индикатор)
   - Средняя зарплата (от/до)
   - Количество вакансий
   - Город
   - Связанные специальности вузов
   - Описание (если есть)
   - Общий рейтинг (excellent/good/average/below_average)

3. **Статистика**:
   - Общее количество профессий
   - Количество с высоким/низким спросом
   - Средняя зарплата
   - Топ-5 профессий по рейтингу

4. **Фильтрация и поиск**:
   - Поиск по названию
   - Фильтр по спросу
   - Фильтр по категории
   - Комбинированные фильтры
